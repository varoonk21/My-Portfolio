# Contributing to My Portfolio

First off, thank you for considering contributing to my portfolio project! It's people like you that make this project
better.

When contributing to this repository, please first discuss the change you wish to make via issue, email, or any other
method with the owner of this repository before making a change.

---

## Development Setup

### Prerequisites

- [Node.js](https://nodejs.org/) (v20 or higher)
- [npm](https://www.npmjs.com/)

### Installation

1. **Clone the repo:**

   ```bash
   git clone https://github.com/varoonk21/My-Portfolio.git
   cd My_portfolio
   ```

2. **Frontend Setup:**

   ```bash
   cd frontend
   npm install
   ```

3. **Backend Setup:**

   ```bash
   cd backend
   npm install
   ```

4. **Start Development Server:**

   ```bash
   cd ..
   npm run dev
   ```

---

## Pull Request Process

1. **Setup & Cleanliness**: Ensure your code is clean and tested before submitting a pull request.
2. **Documentation**: Update the `README.md` with details of changes to the interface or project structure if necessary.
3. **Format Code**: To ensure consistent formatting across the project, run the prettier formatter if you have changed
   any files:
   ```bash
   npm run format
   ```
4. **Approval**: You may request a review for your Pull Request, and once approved, it will be merged into the main
   branch.

## Code of Conduct

### Our Pledge

We as contributors and the maintainer pledge to making participation in our project and our community a harassment-free
experience for everyone, regardless of age, body size, disability, ethnicity, gender identity and expression, level of
experience, nationality, personal appearance, race, religion, or sexual identity and orientation.

### Our Standards

Examples of behavior that contributes to creating a positive environment include:

- Using welcoming and inclusive language.
- Being respectful of differing viewpoints and experiences.
- Gracefully accepting constructive criticism.
- Focusing on what is best for the community.
- Showing empathy towards other community members.

### Enforcement

Instances of abusive, harassing, or otherwise unacceptable behavior may be reported by contacting the project owner. All
complaints will be reviewed and investigated.
